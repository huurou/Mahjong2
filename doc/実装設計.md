# 牌

- Tileレコード
- Tileをすべての牌の基底レコードとし、各牌のレコードを定義する

```
Tile
 ├─NumberTile
 │  ├─ManTile
 │  │  ├─Man1
 │  │
 │  ├─PinTile
 │  │  ├─Pin1
 │  │
 │  └─SouTile
 │     ├─Sou1
 │
 └─HonorTile
    ├─WindTile
    │  ├─Ton
    │  
    └─DragonTile
       ├─Haku
```
